# Cessation

Cessation is a script that initiates the termination of startup processes.

## Usage

| Command    | Description                             |
|       ---: | :---                                    |
| `schedule` | Plans for the termination of a process. |
| `display`  | Shows processes ready for cessation.    |
| `remove`   | Deletes the startup script.             |
| `exit`     | Terminates the script.                  |
